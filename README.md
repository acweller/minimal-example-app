# Minimal Function App Vercel

![Badge Status](https://img.shields.io/badge/status-active-green)

## Descrição
Exemplo básico de um serviço Web com uma Function Serverless para o Vercel.

## Instalação e execução
1. Clone o repo.
    ```bash
    git clone https://github.com/acweller/minimal-example-app.git
    ```
2. Instale dependências.
    ```bash
    npm install
    ```
3. Execute o projeto
    ```bash
    npm run dev
    ```
